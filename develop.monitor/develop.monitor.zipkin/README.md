## 一、ZipKin安装
* 下载zipkin-server-2.12.9-exec.jar

    ```wget https://search.maven.org/remote_content?g=io.zipkin.java&a=zipkin-server&v=LATEST&c=exec```
* 运行zipkin-server-2.12.9-exec.jar

    ```java -jar zipkin-server-2.12.9-exec.jar```
